# portfolio-prototypes

Interactive product-design prototypes behind the case studies on **[siarheimardovich.com](https://siarheimardovich.com)**, by **Siarhei Mardovich**, Principal Product Designer (Fintech, Capital Markets & Decision Intelligence).

Each prototype is a functional, data-rich interface, not a static mockup. They demonstrate decision-intelligence and risk-visualization patterns built in a production front-end stack. I use React and TypeScript with AI coding agents to turn design strategy into working logic, so engineering teams start from something real.

## Stack

React · TypeScript · Vite · Tailwind CSS · Recharts · D3.js · Leaflet · WebGL. Single-file and small-build prototypes, zero heavy dependencies.

## Run locally

```bash
# any static server works; for single-file demos just open the file
cd docs/volatility-surface && python3 -m http.server 8080
```
